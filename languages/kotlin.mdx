## Overview

Kotlin is a modern, statically-typed programming language that runs on the JVM and is Google's preferred language for Android development. This cheat sheet covers key Kotlin concepts.

## Getting Started

### Main Function

```kotlin
fun main() {
    println("Greetings, CheatSheets.zip!")
    // Code goes here
}
```

<Note>
The `main()` function is the starting point of every Kotlin program.
</Note>

## Variables

### Mutable vs Immutable

<CodeGroup>

## Control Flow

### If Expression

<CodeGroup>

```kotlin Basic
var morning = true

if (morning) {
    println("Rise and shine!")
}
```

```kotlin If-Else-If
var age = 65

if (age < 18) {
    println("You are a minor")
} else if (age < 60) {
    println("You are an adult")
} else {
    println("You are a senior")
}
```

</CodeGroup>

### When Expression

```kotlin
var weather = "rainy"

when (weather) {
    "sunny" -> println("Get sunscreen")
    "rainy" -> println("Take an umbrella")
    "snowing" -> println("Wear snow boots")
    else -> println("Invalid weather")
}
```

### Range Operators

```kotlin
val range1 = 1...10      // 1 to 10 (inclusive)
val range2 = 1..<10      // 1 to 9 (exclusive)
val range3 = ...10       // Everything up to 10
```

### While Loop

```kotlin
var counter = 1
var stopNum = 10

while (counter < stopNum) {
    println(counter)
    counter += 1
}
```

### Loop Control

<CodeGroup>

```kotlin Continue
for (num in 0...5) {
    if (num % 2 == 0) {
        continue
    }
    println(num)
}
// Output: 1 3 5
```

```kotlin Break
for (char in "supercalifragilistic") {
    if (char == 'c') {
        break
    }
    println(char)
}
// Output: s u p e r
```

</CodeGroup>

### Return Statement

```kotlin
fun getArea(length: Int, width: Int): Int {
    var area = length * width
    return area
}

var myArea = getArea(10, 8)
println("The area is $myArea.")  // The area is 80.
```

### Single Expression Functions

```kotlin
fun fullName(firstName: String, lastName: String) = "$firstName $lastName"

println(fullName("Ariana", "Ortega"))  // Ariana Ortega
```

## Classes

### Basic Class

```kotlin
class Student {
    var name = "Lucia"
    var semester = "Fall"
    var gpa = 3.95
}

fun main() {
    var student = Student()
    println(student.name)  // Lucia
}
```

### Primary Constructor

```kotlin
class Student(
    val name: String,
    val gpa: Double,
    val semester: String,
    val graduationYear: Int
)

fun main() {
    var student = Student("Lucia", 3.95, "Fall", 2022)
    println(student.name)  // Lucia
}
```

### Member Functions

```kotlin
class Student(val name: String, val gpa: Double) {
    fun calculateLetterGrade(): String {
        return when {
            gpa >= 3.0 -> "A"
            gpa >= 2.7 -> "B"
            gpa >= 1.7 -> "C"
            else -> "F"
        }
    }
}

val student = Student("Lucia", 3.95)
println(student.calculateLetterGrade())  // A
```

## Null Safety

### Nullable Types

```kotlin
var a: String = "Kotlin"  // Cannot be null
// a = null  // Compilation error

var b: String? = "Kotlin" // Can be null
b = null  // OK
```

### Safe Calls

```kotlin
val b: String? = null
println(b?.length)  // Returns null if b is null
```

### Elvis Operator

```kotlin
val l = b?.length ?: -1  // Returns -1 if b is null
```

## Key Features

<Accordion title="Extension Functions">
  Add new functions to existing classes:
  
  ```kotlin
  fun String.removeFirstLastChar(): String {
      return this.substring(1, this.length - 1)
  }
  
  println("Hello".removeFirstLastChar())  // ell
  ```
</Accordion>

  ```kotlin

  val user = User("John", 30)
  println(user)  // User(name=John, age=30)
  ```
</Accordion>
