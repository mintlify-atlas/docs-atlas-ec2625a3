## Overview

Java is a class-based, object-oriented programming language designed to have minimal implementation dependencies. This cheat sheet covers fundamental Java concepts from the mobile app.

## Getting Started

### Hello World

```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, world!");
    }
}
```

**Compiling and Running:**

```bash
$ javac Hello.java
$ java Hello
Hello, world!
```

## Control Flow

### If-Else Statements

```java
int score = 99;

if (score == 100) {
    System.out.println("Superb");
} else if (score >= 90) {
    System.out.println("Excellent");
} else if (score >= 80) {
    System.out.println("Very Good");
} else {
    System.out.println("Good");
}
```

### Switch Statement

```java
int month = 3;
String str;

switch (month) {
    case 1:
        str = "January";
        break;
    case 2:
        str = "February";
        break;
    case 3:
        str = "March";
        break;
    default:
        str = "Some other month";
        break;
}

System.out.println("Result: " + str);  // Result: March
```

### Ternary Operator

```java
int a = 10;
int b = 20;
int max = (a > b) ? a : b;

System.out.println(max);  // 20
```

## Classes and Objects

### Defining Classes

```java
class MyClass {
    int myNum;
    String myString;
    
    // Constructor
    MyClass() {
        myNum = 0;
        myString = "";
    }
    
    // Method
    void myMethod() {
        System.out.println("Hello World!");
    }
}
```

### Creating Objects

```java
MyClass myObj = new MyClass();

myObj.myNum = 15;
myObj.myString = "Hello";

System.out.println(myObj.myNum);     // 15
System.out.println(myObj.myString);  // Hello

myObj.myMethod();  // Hello World!
```

## Collections

### ArrayList

```java
import java.util.ArrayList;

ArrayList<String> cars = new ArrayList<String>();
cars.add("Volvo");
cars.add("BMW");
cars.add("Ford");

System.out.println(cars);  // [Volvo, BMW, Ford]
System.out.println(cars.get(0));  // Volvo
cars.remove(0);
System.out.println(cars);  // [BMW, Ford]
```

### HashMap

```java
import java.util.HashMap;

HashMap<String, Integer> ages = new HashMap<String, Integer>();
ages.put("John", 30);
ages.put("Jane", 25);

System.out.println(ages.get("John"));  // 30
System.out.println(ages.size());       // 2
```

## Exception Handling

### Try-Catch

```java
try {
    int[] myNumbers = {1, 2, 3};
    System.out.println(myNumbers[10]);
} catch (Exception e) {
    System.out.println("Something went wrong.");
} finally {
    System.out.println("The try-catch is finished.");
}
```

## Key Features

<Accordion title="Inheritance">
  Extend classes to create specialized versions:
  
  ```java
  class Animal {
      void sound() {
          System.out.println("Animal makes a sound");
      }
  }
  
  class Dog extends Animal {
      void sound() {
          System.out.println("Dog barks");
      }
  }
  
  Dog dog = new Dog();
  dog.sound();  // Dog barks
  ```
</Accordion>

<Accordion title="Interfaces">
  Define contracts that classes must implement:
  
  ```java
  interface Animal {
      void sound();
  }
  
  class Dog implements Animal {
      public void sound() {
          System.out.println("Bark");
      }
  }
  ```
</Accordion>

<Tip>
Java follows the principle of "Write Once, Run Anywhere" (WORA) through its platform-independent bytecode.
</Tip>
