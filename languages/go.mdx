## Overview

Go (Golang) is a statically-typed, compiled language designed for simplicity and efficiency. This cheat sheet covers essential Go concepts from the mobile app.

## Getting Started

### Hello World

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, world!")
}
```

**Running:**

```bash
$ go run hello.go
Hello, world!
```

### Variables

<CodeGroup>

```go Explicit Declaration
var s1 string
s1 = "Learn Go!"

var b, c int = 1, 2
var d = true
```

```go Short Declaration
s1 := "Learn Go!"  // string
b, c := 1, 2       // int
d := true          // bool
```

</CodeGroup>

<Note>
Short declaration (`:=`) can only be used inside functions.
</Note>

```go

```

```go

    fmt.Println("Other")
}
```

## Functions

### Basic Functions

```go
func plus(a int, b int) int {
    return a + b
}

result := plus(1, 2)
fmt.Println(result)  // 3
```

### Multiple Returns

```go
func vals() (int, int) {
    return 3, 7
}

a, b := vals()
fmt.Println(a)  // 3
fmt.Println(b)  // 7
```

### Variadic Functions

```go
func sum(nums ...int) int {
    total := 0
    for _, num := range nums {
        total += num
    }
    return total
}

fmt.Println(sum(1, 2))     // 3
fmt.Println(sum(1, 2, 3))  // 6
```

## Structs and Maps

### Structs

```go
type Vertex struct {
    X int
    Y int
}

v := Vertex{1, 2}
v.X = 4
fmt.Println(v.X, v.Y)  // 4 2
```

### Maps

```go
m := make(map[string]int)
m["k1"] = 7
m["k2"] = 13

fmt.Println(m)        // map[k1:7 k2:13]
fmt.Println(m["k1"])  // 7
fmt.Println(len(m))   // 2

delete(m, "k2")
fmt.Println(m)        // map[k1:7]
```

## Concurrency

### Goroutines

```go
func f(from string) {
    for i := 0; i < 3; i++ {
        fmt.Println(from, ":", i)
    }
}

func main() {
    f("direct")
    go f("goroutine")  // Run concurrently
    
    go func(msg string) {
        fmt.Println(msg)
    }("going")
}
```

### Channels

```go
messages := make(chan string)

go func() {
    messages <- "ping"  // Send
}()

msg := <-messages  // Receive
fmt.Println(msg)   // ping
```

## Pointers

### Basic Pointers


## Interfaces

### Defining Interfaces

```go
type Shape interface {
    Area() float64
    Perimeter() float64
}

type Rectangle struct {
    Length, Width float64
}

func (r Rectangle) Area() float64 {
    return r.Length * r.Width
}

func (r Rectangle) Perimeter() float64 {
    return 2 * (r.Length + r.Width)
}
```

## Error Handling

### Errors

```go
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}

result, err := divide(10, 0)
if err != nil {
    fmt.Println("Error:", err)
} else {
    fmt.Println("Result:", result)
}
```

## Web Servers

### HTTP Server (Standard Library)

```go
package main

import (
    "fmt"
    "net/http"
)

func handler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprint(w, "Hello from Go!")
}

func main() {
    http.HandleFunc("/", handler)
    http.ListenAndServe(":8080", nil)
}
```

### Using Gin Framework

```go
package main

import "github.com/gin-gonic/gin"

func main() {
    r := gin.Default()
    
    r.GET("/", func(c *gin.Context) {
        c.String(200, "Hello from Gin!")
    })
    
    r.Run(":8080")
}
```

## Key Features

<Accordion title="Defer">
  Execute code after function returns:
  
  ```go
  func main() {
      defer fmt.Println("Done")
      fmt.Println("Working...")
  }
  // Output:
  // Working...
  // Done
  ```
</Accordion>

<Accordion title="JSON Handling">
  Encode and decode JSON:
  
  ```go
  type Person struct {
      Name string `json:"name"`
      Age  int    `json:"age"`
  }
  
  // Encoding
  data, _ := json.Marshal(person)
  
  // Decoding
  json.Unmarshal(data, &person)
  ```
</Accordion>

Go's concurrency model with goroutines and channels makes it excellent for building scalable network services.
