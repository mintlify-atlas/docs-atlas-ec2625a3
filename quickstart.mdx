## First Launch

When you first open the Cheat Sheets app, you'll see the main browse screen with all available cheat sheets organized by category.

<Steps>
  <Step title="Browse Cheat Sheets">
    The home screen displays all available cheat sheets as colorful cards. Each card shows:
    - **Language/Tool name** (e.g., Python, Git, Docker)
    - **Category tag** (Programming, Toolkit, Other)
    - **Brief description**
    
    Scroll through the list to explore what's available.
  </Step>

  <Step title="Select a Cheat Sheet">
    Tap any cheat sheet card to open it. You'll see:
    - **Introduction** with a brief overview
    - **Sections** organized by topic
    - **Code examples** with syntax highlighting
    - **Copy buttons** to quickly copy code snippets
  </Step>

  <Step title="Navigate Sections">
    Each cheat sheet is divided into logical sections. Use the section list to jump directly to the topic you need:
    
    - Getting Started
    - Syntax & Data Types
    - Control Flow
    - Functions & Methods
    - Advanced Topics
    - Tips & Tricks
  </Step>
</Steps>

## Using Search

The search feature helps you find specific commands, syntax, or topics across all cheat sheets.

<Steps>
  <Step title="Open Search">
    Tap the **search icon** in the top navigation bar to open the search screen.
  </Step>

  <Step title="Enter Query">
    Type your search query. The app will search through:
    - Cheat sheet titles
    - Content within cheat sheets
    - Code examples
    - Section headings
    
    <Accordion title="Search Examples">
      Try searching for:
      - `"for loop"` - Find loop syntax across languages
      - `"git commit"` - Find Git commit commands
      - `"array"` - Find array operations in different languages
      - `"async"` - Find async/await patterns
    </Accordion>
  </Step>

  <Step title="View Results">
    Search results show matching cheat sheets with highlighted text. Tap any result to open that cheat sheet at the relevant section.
  </Step>
</Steps>

## Customizing the App

Make the app your own by adjusting settings to match your preferences.

### Theme Selection

Choose between Light, Dark, or System theme:

1. Tap the **settings icon** (gear) in the top navigation
2. Select **Theme**
3. Choose your preferred option:
   - **Light** - Bright background, easier in daylight
   - **Dark** - Dark background, easier on the eyes at night
   - **System** - Follows your device's system theme

<Tip>
  Dark mode is perfect for late-night coding sessions and reduces eye strain.
</Tip>

### Other Settings

<CardGroup cols={2}>
  <Card title="Font Size" icon="text-size">

  </Card>
  <Card title="Categories" icon="filter">

  </Card>
</CardGroup>

## Common Use Cases

### Learning a New Language

<Steps>
  <Step title="Open Language Cheat Sheet">
    Navigate to the language you want to learn (e.g., Python, Rust, JavaScript).
  </Step>

  <Step title="Start with Getting Started">
    Read the Getting Started section for a Hello World example and basic syntax.
  </Step>

  <Step title="Practice with Examples">
    Copy code examples and run them in your IDE or code editor. Modify them to understand how they work.
  </Step>
</Steps>

### Quick Reference During Development

When you need to look up syntax quickly:

1. Open the app on your phone or tablet
2. Search for the specific command or syntax
3. Copy the code snippet
4. Paste it into your development environment

<Tip>
  Keep the app open on a second screen or phone while coding for instant access to reference material.
</Tip>

### Interview Preparation

Prepare for technical interviews by reviewing:

- **Data Structures** - Arrays, lists, maps, sets
- **Algorithms** - Sorting, searching, recursion
- **Common Patterns** - Loops, conditionals, error handling
- **Language-Specific** - Unique features of each language
