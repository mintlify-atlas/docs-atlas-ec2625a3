Cheat Sheets is a comprehensive mobile reference guide designed to help developers quickly access programming language syntax, commands, and best practices. Built with Flutter, the app provides a beautiful, cross-platform experience for browsing curated cheat sheets contributed by the open source community.

## What is Cheat Sheets?

Cheat Sheets is a mobile application that gives developers instant access to quick reference guides for popular programming languages and tools. Whether you're learning a new language or need a quick syntax reminder, Cheat Sheets provides well-organized, searchable content right at your fingertips.

The app fetches content from a curated collection of community-contributed cheat sheets, making it a living resource that grows with the developer community.

## Supported Platforms

Cheat Sheets is available on both major mobile platforms:

<Tabs>
  <Tab title="Android">
    - Android 5.0 (API level 21) and above
    - Available on Google Play Store
    - Direct APK download from GitHub Releases
  </Tab>
  <Tab title="iOS">
    - iOS 12.0 and above
    - Distributed via AltStore (no jailbreak required)
    - IPA files available on GitHub Releases
  </Tab>
</Tabs>
